# Nickname-Generator

```sh
-h --help                Show help message
-n --num  (97+)          Specify the number, which sum of ASCII characters weight should be equal to
-m --min  (1+)           Minimum length of one word in nickname
-x --max  (1+)           Maximum length of one word in nickname
-l --l33t (0.0 - 1.0)    Make characters 1337 in percent
-s --seed (any number)   If you need it
-f --first(any char)     Specify the first character
-an --adj_noun  flag     Make nickname using 1 adjective and 1 noun
-vn --verb_noun flag     1 verb and 1 noun
-nn --noun_noun flag     2 nouns
```

Press "Enter" to show next five words

Press "Ctrl + C" to exit
